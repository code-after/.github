## Code After Series Overview

Code After is a research programme examining how artificial intelligence reshapes the institutional structures of everyday life. Version 0.9 provides the vocabulary and methodologies for understanding AI’s impact on law, accounting, and governance. The Series extends this foundation through six papers, each addressing one domain—language, education, work, evidence, measurement, and jurisdiction—and explaining how AI reorganizes the systems societies rely on to coordinate action, establish trust, and legitimate authority. The programme also develops a multilingual framework to enable meaningful AI debate in the native languages of global citizens.
